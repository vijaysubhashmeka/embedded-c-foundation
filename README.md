# Embedded C Foundation

This repository contains resources for learning Embedded C, structured into various sections covering different fundamental concepts and projects.

## Directory Structure

- **01-basics**: Basic concepts of Embedded C.
- **02-control-flow**: Control flow structures in Embedded C.
- **03-functions**: Understanding functions and their usage.
- **04-arrays-strings**: Working with arrays and strings.
- **05-pointers**: Introduction to pointers and memory management.
- **06-structures**: Understanding structures and their applications.
- **07-projects**: Practical projects to apply the knowledge gained.
